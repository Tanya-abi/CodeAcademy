Truthy and Falsy

In JavaScript, truthy and falsy values determine how a value behaves in a Boolean context (e.g., in `if` statements). Truthy values are those that evaluate to `true` in a Boolean context, such as non-empty strings, non-zero numbers, and objects. Falsy values, on the other hand, evaluate to `false`. Common falsy values include `0`, `""` (empty string), `null`, `undefined`, `NaN`, and `false`. You can use these inherent properties without explicitly checking for `true` or `false` in conditions.
