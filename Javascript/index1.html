Else if Statements

The else if statement allows you to check multiple conditions in a sequence. When the first if condition is not true, the program checks the next else if condition, and so on. If none of the conditions are true, the else block is executed (if present)

const age = 20;

if (age < 18) {
console.log("You are a minor.");
} else if (age >= 18 && age < 21) {
console.log("You are an adult, but cannot drink alcohol.");
} else {
console.log("You are an adult and can drink alcohol.");
}
const temperature = 35;

if (temperature < 10) {
console.log("It's freezing outside.");
} else if (temperature < 25) {
console.log("The weather is mild.");
} else {
console.log("It's hot outside.");
}
